Supported codecs

  • Ogg Vorbis
  • Ogg Opus
  • FLAC (including OggFLAC)
  • AC3 (including E-AC3)
  • DTS (including DTS HD-MA)
  • TrueHD
  • AAC
  • USAC
  • MP3
  • WAV
  • AIFF
  • Musepack
  • Monkey's Audio
  • APL (APE Linking File)
  • Shorten
  • WavPack
  • Apple Lossless (ALAC)
  • WMA Standard, Pro, Lossless, and Voice
  • TrueAudio
  • TAK (Tom's Lossless Audio Kompressor)
  • TwinVQ
  • RealAudio
  • ATRAC3
  • ATRAC3Plus
  • Direct Stream Digital (DSD)
  • Interchage File Format (IFF)
  • MIDI Sequences in various formats
  • Sequenced Module formats (IT, XM, S3M, MOD, STM, PTM, MTM, 669, PSM, AM, J2B, DSM, AMF, OKT/OKTA, UMX, etc. See: libOpenMPT)
  • Emulated Console formats supported by the Game_Music_Emu library (AY, GBS, HES, KSS, NSF/NSFE, SAP, SGC, SPC, and VGM/VGZ)
  • SID format supported by sidplayfp
  • Many Emulated Console formats utilizing the PSF format (PSF, PSF2, SSF, DSF, QSF, GSF, NCSF, 2SF, and their respective mini variants)
  • Any codec supported by Mac OS X's Core Audio API

Supported tags

  • Vorbis comments/FLAC tags
  • ID3v1.0, 1.1, 2.3+
  • APEv1 and v2
  • RIFF INFO chunks (and ID3v2 in `id3 ` chunks)

Supported Archive formats

  • ZIP
  • GZip
  • RAR (also renamed to RSN)
  • 7-Zip (also renamed to VGM7Z)

General features

  • Notification support
  • M3U and PLS playlist
  • HTTP/S streaming
  • HTTP/S Live Streaming (HLS) of FFmpeg supported formats
  • Output device selection
  • Gapless playback
  • Logarithmic volume control
  • Smart shuffle
  • Seeking
